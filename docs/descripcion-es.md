# Descripción del proyecto — CursorTrail

## Resumen

**CursorTrail** es una pequeña aplicación nativa para macOS que añade una estela suave detrás del cursor del ratón.

La aplicación vive en la barra de menús, funciona de forma local y no necesita cuentas, servidores, tokens ni bibliotecas externas.

## Funciones

- estilos Classic, Glow y Dots;
- longitud Short, Medium y Long;
- opacidad ajustable;
- activación y desactivación desde la barra de menús;
- soporte para varias pantallas;
- preferencias guardadas con UserDefaults;
- compilación mediante un script shell.

## Tecnologías

- Swift
- AppKit / Cocoa
- QuartzCore
- NSStatusItem
- ventana transparente superpuesta
- dibujo personalizado

## Privacidad

CursorTrail no utiliza analítica, APIs externas, cuentas ni servicios en la nube. Toda la configuración se guarda localmente.

## Valor del proyecto

Este repositorio demuestra desarrollo nativo para macOS, renderizado personalizado, manejo de eventos, aplicaciones de barra de menús y empaquetado local.
