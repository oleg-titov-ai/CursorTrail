# Changelog

All notable changes to CursorTrail are documented in this file.

The format is based on Keep a Changelog, and the project follows semantic versioning where practical.

## [1.0.0] - Initial portfolio release

### Added

- Native Swift menu-bar application.
- Classic, Glow, and Dots trail styles.
- Short, Medium, and Long trail lengths.
- Adjustable opacity.
- Multi-display support.
- Local settings persistence with UserDefaults.
- Shell-based build script.
- Ad-hoc local code signing.
