# Support

CursorTrail is maintained as a small public portfolio project.

## Getting Help

Before opening an issue:

1. read the README;
2. check the usage guide;
3. build the app locally;
4. test the app in normal desktop windows;
5. check whether the issue only happens in fullscreen apps.

## What to Include in an Issue

- short problem description;
- steps to reproduce;
- macOS version;
- selected trail style;
- selected trail length;
- selected opacity;
- whether the problem happens on one display or multiple displays.

## What Not to Include

Do not include private screenshots, Apple Developer credentials, signing certificates, personal system logs, or unrelated files.

## Known Limitation

Some fullscreen apps and protected macOS surfaces may draw above the CursorTrail overlay.

## Project Status

This is not a commercial support channel. Issues and improvements are handled as time allows.
