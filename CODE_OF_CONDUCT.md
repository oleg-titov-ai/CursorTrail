# Code of Conduct

## Our Standard

CursorTrail is maintained as a small public portfolio project. Discussions should stay respectful, practical, and focused on improving the app.

Examples of good behavior:

- clear bug reports;
- constructive pull request feedback;
- respectful technical disagreement;
- safe screenshots without private information;
- useful documentation and usability suggestions.

## Unacceptable Behavior

Unacceptable behavior includes:

- harassment or personal attacks;
- publishing private screenshots, certificates, credentials, or personal system logs;
- spam;
- abusive language;
- irrelevant promotion;
- requests to misuse macOS permissions or bypass platform security.

## Enforcement

Maintainers may edit, close, delete, or lock issues and pull requests that violate this code of conduct.

## Reporting

Report problems through GitHub issues with a safe, factual description. Do not include private files, certificates, credentials, or personal information.
